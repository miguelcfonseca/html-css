Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E neste parágrafo, temos outro texto marcado no final.

A cor padrão da marcação é amarelo. Para mudar a cor podemos dentro da tag MARK adicionar um parâmetro style="background-color:color; e escolher a cor que desejamos. Podemos ainda dentro da tag HEAD adicionar uma tag STYLE e definir a mesma cor para todas as tags mark dessa forma
mark {
background-color:color;
}
Esta forma é mais vantajosa pois aplica essa cor para todas as tag mark. Mais à frente, estudaremos ainda outra forma de adicionar CSS mas de forma externa
