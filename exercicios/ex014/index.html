Capítulo 1

Capítulo 1.1

Tudo mais organizado! Agora, o conteúdo está separado do estilo dentro do código já que o conteúdo está em <body> e </body> e o estilo entre <head> e </head>. Também é menos trabalhosa esta forma. Enquanto que com estilos inline, tínhamos que escrever 3 linhas de CSS para colocar três h2 da mesma cor, agora usando um seletor, a propriedade aplica-se a todos os h2.

Capítulo 1.2

Mas também há desvantagens. Ao longo do tempo iremos perceber que CSS ocupa muitas mais linhas do que HTML ou seja quando você abrir um arquivo de código, vai existir um estilo gigante até você finalmente chegar no conteúdo. Vamos ver a outra! Neste caso nós temos apenas uma página, o index.html. O estilo que escrevemos entre <head> e </head> serve apenas para esta página. Se o seu cliente tiver um site com 20 páginas e quiser aplicar o mesmo estilo nas 20, vai dar trabalho copiar e colar para todas as páginas.
